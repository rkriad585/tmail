# Contributing

Thanks for your interest in contributing to TempMail!

## Getting started

```bash
pip install uv
uv sync
tmail
```

## Guidelines

- Keep changes in `mail.py` — this is a single-file application.
- Run `python mail.py` to manually test before submitting.
- No tests or test runner is configured yet.
- Follow the existing code style (no type annotations, no docstring conventions enforced).

## Pull requests

- Open an issue first to discuss significant changes.
- Keep PRs focused on a single concern.
- Update `README.md` if adding or changing user-facing features.
